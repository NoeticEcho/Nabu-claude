# /nabu-consolidate

Запусти консолидацию памяти субагентами `reflector` и `librarian`:

1. **Рефлексия** (`reflector`): `nabu-memory.list_recent_episodes` → сгруппировать по периоду →
   синтезировать связный нарратив → `nabu-memory.save_narrative({ period, narrative })`.
   Устойчивые повторяющиеся паттерны → `nabu-memory.add_fact`.
2. **Гигиена** (`librarian`): выявить дубли/устаревшее/слабо связанное; предложить объединение и
   связи в графе. Ничего не удалять без явного согласия пользователя.
3. **Процедуры**: где есть исходы — обновить процедурную память.

Тяжёлые операции (эмбеддинги) — локально. private/vault — только локально. Не выдумывать —
нарратив и факты строго из записанных эпизодов. Wellbeing (SAFETY.md): бережно, без руминации.
