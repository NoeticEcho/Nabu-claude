# /nabu-digest

Собери дайджест за `$ARGUMENTS` (по умолчанию неделя) субагентом `digest`:
события (`nabu-memory.list_recent_episodes`), темы (`nabu-memory.recall`), открытые намерения
(`nabu-memory.list_prospective`). Структура: Главное · Прогресс · Открытое · Инсайты. Только из
памяти, не выдумывать; private/vault локально; бережно (docs/28). Для долговременной консолидации —
субагент `reflector`.
