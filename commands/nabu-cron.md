# /nabu-cron

Помоги пользователю включить проактивные задачи Nabu по расписанию. Плагин не может напрямую
поставлять cron, поэтому используем механизмы Claude Code (см. `docs/SCHEDULING.md`):

- **Облачные routines** (`/schedule`) — работают без открытой сессии, минимум 1 час. Рекомендуется
  для регулярных фоновых задач.
- **Сессионный `/loop`** — пока открыта сессия, интервал от 1 мин.
- **Системный cron** (опц.) — `scripts/install-cron.sh` регистрирует headless-запуски.

## Каталог проактивных задач Nabu
| Задача | Что делает | Рекоменд. частота | Команда/промпт |
|---|---|---|---|
| consolidate | эпизоды → нарратив, гигиена памяти | ежедневно | `/nabu-consolidate` |
| digest | сводка за период | ежедневно/еженедельно | `/nabu-digest неделя` |
| research | ищет улучшения Nabu/агентов (фаза C) | еженедельно | `/nabu-research` |
| scout | новые скиллы/MCP/скрипты (фаза C) | еженедельно | `/nabu-scout` |
| feedback | опрос как применён совет (фаза D) | 2-3×/нед | `/nabu-feedback` |
| metrics | сбор/корреляция метрик жизни (фаза D) | ежедневно | `/nabu-metrics` |

## Действия
- **список**: `CronList` + показать активные + этот каталог.
- **enable `<job>`**: предложи точную команду `/schedule "<cron|описание>" <промпт задачи>` (облако) или
  `/loop <интервал> <промпт>` (сессия); при желании — `scripts/install-cron.sh <job> "<cron>"`.
- **disable `<job>`**: `CronDelete` по id или удалить из системного crontab.

Зафиксируй включённые задачи как `system_task(kind=scheduled)` через `nabu-memory.add_system_task`,
чтобы Nabu знал о своём расписании. Приватное обрабатывается локально и при облачных routines.
