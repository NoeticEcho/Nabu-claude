# /nabu-voice

Транскрибируй `$ARGUMENTS` через voice-server (= Transcriber #06): локальный Whisper
(faster-whisper). Диаризация — опционально (config.features.diarization, нужен HF-токен).
Результат → эпизодическая память + прогон через конвейер (entity-extractor и т.д.).
Неблокирующее: при недоступности модели — сообщить и продолжить без блокировки.
