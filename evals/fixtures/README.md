# Фикстуры eval (записанные реальные ответы агентов)

Пусто по умолчанию. Заполняется `npm run eval:record` (диспатч агентов вживую + LLM-судья +
запись ответов сюда как `<set>/<id>.txt` и baseline в `../baselines/`). Служат для
детерминированной регрессии в CI без повторных LLM-вызовов.

ВАЖНО: офлайн-replay фикстур БЕЗ `--judge` использует только подстрочный гейт (границы слов),
который слеп к отрицанию/цитированию — для граничных/кризисных наборов он даёт ложные срабатывания.
Авторитетная проверка таких наборов — с `--judge` (семантика). См. `../README.md`.
