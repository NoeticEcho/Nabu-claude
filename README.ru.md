# Nabu — ваш личный ИИ-совет

> Совет ИИ-министров по всем сферам жизни — здоровье, деньги, работа, отношения,
> рост — с **памятью на всю жизнь**, работающий **на вашей машине**, на той
> подписке Claude, которая у вас уже есть.

Nabu — это не ещё один чат-бот, который забывает вас через сообщение. Это **команда
специализированных ИИ-агентов**: 9 доменных «министров», специалисты для предпринимателя и в сумме 74 агента
обработки, которые видят вашу жизнь целиком, помнят всё, что вы решили им доверить,
коллегиально обсуждают сложные вопросы и отвечают с честными trade-off'ами, а не
усреднённой кашей.

- 🧠 **Мозг — Claude Code.** Рассуждает ваша существующая подписка Claude.
  Никаких дополнительных API-ключей и потокенной оплаты.
- 🏠 **Данные остаются дома.** Postgres + pgvector + TypeDB — в локальном Docker;
  эмбеддинги считает локальная модель (Ollama). Всё ставится одной командой.
- 🎖 **Адъютант, а не приложение.** Общайтесь в браузере или Telegram (текст и голос);
  сложные вопросы созывают Совет; каждое рискованное действие ждёт **вашей** кнопки.

## Быстрый старт

```bash
# Нужны: Node ≥22, Docker, Claude Code CLI (подписка Max), ~5 минут
curl -fsSL https://raw.githubusercontent.com/noeticecho/nabu-claude/master/scripts/install.sh | bash

nabu start   # демон: расписание + веб-чат + (опц.) Telegram-бот
nabu chat    # открыть http://127.0.0.1:4517
```

Windows (PowerShell):

```powershell
irm https://raw.githubusercontent.com/noeticecho/nabu-claude/master/scripts/install.ps1 | iex
```

`nabu init` — zero-config: генерирует `.env` со случайными паролями, подбирает
свободные порты, поднимает Docker-стек (pgvector + TypeDB + Ollama при отсутствии),
применяет схемы, докачивает embedding-модель и прогоняет smoke-тест. Идемпотентно.

## Приватность как архитектура

- **Local-first.** Вся память — в ваших Docker-томах; всё слушает только `127.0.0.1`,
  веб-сервер отвергает DNS-rebinding.
- **Три уровня видимости:** `default` · `private` (по умолчанию для здоровья/денег/
  отношений) · `vault`.
- **Vault зашифрован E2E.** AES-256-GCM машинным ключом **до** записи в БД. Для vault
  не считаются даже локальные эмбеддинги, а его текст не попадает в контекст модели
  обычными путями: разбор vault-заметок делает локальная LLM (`extract_entities_local`).
- **Риск подтверждаете вы.** Любое внешнее/финансовое/деструктивное действие создаёт
  запрос, который закрывает только **человеческая кнопка** (веб ✅/❌ или inline-клавиатура
  Telegram) — модель не может одобрить сама себя.
- **Честные логи.** JSONL-логи хранят тайминги и стоимость — никогда текст сообщений.
- **Границы, а не врачи.** Доменные агенты дают информацию и структуру и направляют
  к лицензированным специалистам для медицинских/юридических/финансовых решений.

## 🌱 Nabu Commons

Инстансы Nabu улучшают проект сообща. С согласия пользователя инстанс замечает
проблемы у себя, предлагает изменения (только агрегаты — никаких личных данных),
а сеть голосует реакциями 👍 и evidence-комментами. Разработчики берут приоритетные
предложения; ревьюит и мержит **только человек-мейнтейнер** — инстансы никогда не
одобряют сами себя. Весь цикл **выключен по умолчанию** (`commons.enabled=false`).
Протокол, правила безопасности и чек-лист мейнтейнера — [docs/COMMONS.md](docs/COMMONS.md).

## Участие

PR приветствуются — см. [CONTRIBUTING.md](CONTRIBUTING.md). Коротко о правилах:

только аддитивные; инварианты приватности не обсуждаются; тесты зелёные
