# Плейбук: позиционирование → ICP → каналы → воронка

## 1. Позиционирование (April Dunford, сжатая форма)

1. Конкурентные альтернативы: что пользователь делает СЕЙЧАС вместо продукта (включая «ничего/Excel»).
2. Уникальные атрибуты: что есть у вас, чего нет у альтернатив (честно, 2-4 пункта).
3. Ценность: во что атрибуты конвертируются для клиента (время/деньги/риск/статус).
4. Кому важнее всего (→ ICP), в какой категории рынка это читается без объяснений.

Тест позиционирования: незнакомый человек за 10 секунд с лендинга отвечает «что это, для кого,
почему не X». Не отвечает → переделывать оффер, не рекламу.

## 2. ICP (Ideal Customer Profile)

- Формула: [роль/сегмент] с [болью] в [контексте], который уже [пытался решить как].
- Признаки хорошего ICP: можно найти 50 таких людей за неделю; боль осознана (ищут решение);
  платёжеспособны; вы можете до них дотянуться существующими каналами.
- Solo-фаундеру: ОДИН ICP до устойчивого сигнала. Второй сегмент = второй продукт по затратам.

## 3. Каналы (порядок для solo, до PMF)

| Канал | Когда работает | Стоимость | Сигнал за |
|---|---|---|---|
| Прямые разговоры/DM (20-50 шт.) | всегда первым | время | дни |
| Сообщества (Reddit/HN/TG/Discord ниши) | есть где обитает ICP | время | 1-2 нед |
| Контент/SEO | боль гуглят; горизонт 3-6 мес | время | месяцы |
| Холодный аутрич (email/LinkedIn) | B2B, чёткий ICP | время+инструменты | 2-4 нед |
| Платная реклама | юнит-экономика сходится, LTV известен | деньги | 1-2 нед |
| Product Hunt / запуски | готов онбординг; см. агента startup-gtm | время | день |

Правило: канал считается проверенным после ~100 контактов ICP с измеренной конверсией, не после
«попробовал пару постов».

## 4. Воронка AARRR (пиратские метрики)

- **Acquisition**: посетители по каналам → метрика: качественный трафик (доля ICP).
- **Activation**: дошли до «ага-момента» → метрика: % завершивших ключевое действие.
- **Retention**: вернулись → КОРОЛЕВСКАЯ метрика для продукта; когорты по неделям.
- **Referral**: привели других → viral coefficient, но не раньше retention.
- **Revenue**: платят → конверсия в платящих, churn.

Диагностика: чинить воронку СНИЗУ вверх (retention → activation → acquisition). Лить трафик в
дырявую воронку — сжигание денег.

## 5. Юнит-экономика (минимум)

- CAC = затраты на канал / новых клиентов с канала (включая ваше время по честной ставке).
- LTV ≈ ARPU × средняя жизнь клиента (мес). Ранняя стадия: LTV неизвестен — оценивать по
  когортам, помечать как оценку (Iron Law 1).
- Здоровье: LTV/CAC ≥ 3 (SaaS-эвристика), payback CAC < 12 мес. Хуже → канал не масштабировать.

## 6. Контент-минимум для solo

1 столп-материал в неделю (глубокий пост/видео по боли ICP) → нарезка в 3-5 коротких форматов.
Дистрибуция важнее объёма: 20% создание / 80% дистрибуция в местах обитания ICP.
