## Принципы (не меняются)

1. **Локальность** — память живёт у пользователя; облако не получает приватного.
2. **Approval вне модели** — рискованное ждёт кнопку человека.
3. **Честность** — деградация объявляется, данные не выдумываются, trade-off'ы наружу.
4. **Zero-config, zero-dep** — одна команда, ноль церемоний.
5. **Совет советует** — решает человек.

## Решения (зафиксировано — не пересматривать без причины)

- **Merge в Commons — только человек.** Автоутверждение PR сетью ИИ — supply-chain-вектор. Навсегда.
- **Файлы-экспорты вместо облачных health/bank API** — приватнее, проще, работает у всех.
- **Без Tailwind/фреймворков** в чате; тёмная тема — единственная.
- **Telegram — опциональный клиент** (транзит через их серверы задокументирован; vault туда не ходит).
- **Headless-агенты всегда с `--mcp-config`**; тяжёлое — локально; схемы — только аддитивно.
- **ruflo-рантайм не внедрять** (рой ≠ коллегиальность); взяты только конвенции.
