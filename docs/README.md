# Документация Nabu

Спецификация проекта. Для AI-разработки релевантны документы в этой папке;
справочные (стратегия, монетизация, hardware) — в `_reference/`.

## Для разработки (читать)

**Фундамент:**
- `01_nabu_concept_v2` — концепция
- `03_nabu_srs` — требования (SRS)
- `05_nabu_sad` — архитектура (SAD)
- `06_nabu_adr` — архитектурные решения
- `13_nabu_eng_standards` — инженерные стандарты
- `15_nabu_option_d` — контракты Option Д (инварианты)
- `26_nabu_mcp_skills` — MCP-серверы и skills
- `27_branching_and_migrations` — branching + миграции (Supabase/TypeDB)
- `28_wellbeing_safety` — политика благополучия и безопасности (therapy)

**Что строить:**
- `02_nabu_functional_matrix` — фазы (7-фазный rollout)
- `04_nabu_backlog` — эпики и user stories
- `07_nabu_data_model` — модель данных
- `08_nabu_api_spec` — спецификация API
- `09_nabu_agent_catalog` — каталог 44 агентов
- `LIBRARY` — библиотека справочных знаний (reference): домены, отличие от личной памяти, дистилляция навыков
- `10_nabu_test_strategy` — стратегия тестирования
- `11_nabu_security` — безопасность и приватность
- `12_nabu_ops_runbook` — операционный runbook
