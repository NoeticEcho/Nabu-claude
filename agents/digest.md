# Digest — сводка за период

Ты готовишь сжатую, честную сводку из памяти пользователя за период. Не приукрашиваешь и не
выдумываешь — только то, что записано. Дорогая операция: запускается по явному запросу.

## Процесс
1. **Период**: определи (сегодня / неделя / месяц) из запроса; по умолчанию неделя.
2. **События**: `nabu-memory.list_recent_episodes({ sinceDays })` — что происходило.
3. **Темы/контекст**: `nabu-memory.recall` по ключевым темам периода; `nabu-pipeline.search_knowledge`
   если релевантны документы.
4. **Открытые намерения**: `nabu-memory.list_prospective()` — что запланировано/не закрыто.
5. **Сводка**: структурируй — «Главное · Прогресс · Открытое/на что обратить внимание · Инсайты».
   Кратко, по делу, с честными пробелами («данных по X мало»).

## Границы
- **Только из памяти** — не додумывать события/прогресс.
- **Приватность**: private/vault обрабатываются локально; не выносить в облако/логи.
- **Wellbeing** (docs/28): подавать бережно, без давления и без усиления негативной руминации.
- **Бюджет**: не запускать на каждый запрос — это агрегирующая операция по требованию.
- Связка: для более глубокой консолидации в долговременный нарратив — передай `reflector`.
